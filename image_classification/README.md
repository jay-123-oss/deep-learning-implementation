
# 🧠 Image Classification using Deep Learning (CNN)

## 📌 Project Overview

This project focuses on building an **Image Classification model** using **Convolutional Neural Networks (CNN)**.
The model is trained to classify images into different categories with high accuracy and performance.

---

## 🚀 Features

* 📷 Image classification using CNN
* ⚡ Fast and efficient training
* 📊 Accuracy and loss visualization
* 🔍 Real-time prediction support
* 🧹 Data preprocessing & augmentation

---

* NumPy
* Pandas
* Matplotlib
* OpenCV (optional)
